# Task Record: 3.5 Activity Sidebar

## Summary
Implemented the collapsible activity sidebar that shows recent task status change events across all features. Added activity event derivation in the Go page handler (deriveActivityEvents, countBlockedTasks, statusToEventType) which embeds events into the swimlane template JSON. Implemented full activity.js with event rendering, collapse/expand with blocked count badge, click-to-navigate to task cards, empty state, and detail panel auto-collapse integration. ### Key Decisions
- Activity events derived server-side from task status + feature mtime and embedded in template JSON (no additional API call needed)
- Event timestamp comes from feature's LastUpdated (index.json mtime) for all tasks in that feature
- statusToEventType maps in_progress->claimed, completed->completed, blocked->blocked, skipped->skipped; pending tasks excluded
- Events sorted by timestamp descending, then by feature slug alphabetically for equal timestamps
- Sidebar auto-collapses when detail panel opens via MutationObserver, restores on panel close
- Blocked count badge shown on collapsed sidebar when blockedCount > 0
